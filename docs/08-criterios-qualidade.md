# 8. Critérios de Qualidade Científica

O sistema deve garantir:

| Critério | Descrição |
|----------|-----------|
| **Reprodutibilidade** | Mesma pergunta + mesmo dado → mesmo resultado |
| **Auditabilidade do SQL** | Todo resultado acompanhado do SQL executado |
| **Transparência do cálculo** | Cálculos feitos no motor (DuckDB), não no texto do LLM |
| **Utilidade para gestão** | Respostas úteis para decisão em saúde pública |

Esses critérios alinham o projeto a boas práticas de pesquisa em saúde digital e governança de dados.
